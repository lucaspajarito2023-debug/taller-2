## 📝 Descripción del Proyecto
Esta es una aplicación web de alto rendimiento diseñada para entusiastas del automovilismo. Permite explorar una base de datos técnica de vehículos de competencia (F1 y Supercars) mediante una interfaz dinámica con estética **Neon-Night**.

### 🛠️ Funcionalidades Clave:
* **Búsqueda Dinámica:** Filtrado en tiempo real de vehículos por nombre o marca.
* **Ficha Técnica:** Visualización de 4 atributos críticos (Motor, Velocidad, Piloto y Escudería).
* **Experiencia Inmersiva:** Efectos de sonido de aceleración al realizar búsquedas y animaciones de carga.
* **Interfaz Adaptativa:** Diseño optimizado para visualizar la telemetría a la derecha y el vehículo a la izquierda.

---

## 🌐 API y Datos
Para este proyecto se utiliza una **Arquitectura de Datos Estática (Mock API)** estructurada en un archivo JSON integrado. 

* **¿Por qué?** Para garantizar que las imágenes de alta resolución (4K) y los datos técnicos de escuderías oficiales sean siempre precisos y no dependan de límites de peticiones externos.

## 🚀 Instrucciones de Ejecución Local

Para poner este bólido en marcha en tu computadora, sigue estos pasos:

1.  **Clonar/Descargar:** Descarga el repositorio en una carpeta local.
2.  **Archivos necesarios:** Asegúrate de tener la siguiente estructura:
    ```text
    ├── index.html
    ├── style.css
    ├── main.js
    └── README.md
    ```
3.  **Lanzamiento:** * Simplemente haz doble clic en el archivo `index.html` para abrirlo en tu navegador.
    * *Recomendación:* Usa la extensión **Live Server** de VS Code para una carga de sonidos más fluida.
